# Spring Boot REST API - Study Project

This repository was created for studying and practicing backend development using **Spring Boot**.  
The main goal is to explore concepts related to **REST API development, layered architecture, and best practices used in real-world applications**.

Instead of keeping all changes in a single line of development, each concept and improvement of the project is implemented in **separate branches**, allowing the evolution of the application to be tracked step by step.

## Objectives

- Practice **Spring Boot fundamentals**
- Implement **RESTful APIs**
- Understand **Layered Architecture**
- Apply **clean code and good design practices**
- Explore tools commonly used in backend development

## Concepts Studied

Some of the concepts explored in this repository include:

- REST Controllers
- DTO Pattern
- Service Layer
- Repository Layer
- JPA / Hibernate
- Exception Handling
- Validation
- Dependency Injection
- Clean Architecture principles
